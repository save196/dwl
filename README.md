# dwl - dwm for Wayland

This is my custom build of [dwl].

It includes the following patches:
- bar
- custom simpleborders
- pertag
- attachfocused
- gaps
- bottomstack
- togglekblayoutandoptions
- kblayout
- namedscratchpads

For more info, check out the official repo.

[dwl]: https://codeberg.org/dwl/dwl
